# 基于深度学习的停车场管理系统vue前端

## Build Setup

```bash
# install dependency
npm install

# develop
npm run dev
```
前端项目地址：https://github.com/huerni/parkingSystem  
后端项目地址：https://github.com/huerni/parkingSystemBack  
车牌识别模块：https://github.com/huerni/Parking_license  

前端采用vue开发，后端采用SpringBoot，车牌识别模块使用yoloV5模型进行训练预测
